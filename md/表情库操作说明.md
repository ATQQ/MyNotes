
## 表情库表初始状态

| 字段1 | 字段2 |
| :---: | :---: |
| role  | null  |
| noisy | null  |

* 前端第一次请求获取表情库中的噪音符号(对话角色操作同理)
  * 响应
  ```js
    ""
  ```
  * 前端操作一番新增了几个噪音符号
    1. 开心 happy
    2. 哭声 cry
    3. 大笑  hahaha
  
  * 保存更改过后的内容(一个按钮)
    
    1. 向后端发送的数据
    ```js
    [
        {
            "key":"开心",
            "value":"happy"
        },
        {
            "key":"哭声",
            "value":"cry"
        },
        {
            "key":"大笑",
            "value":"hahaha"
        }
        
    ]
    ```
**后端把这个存进表中**
